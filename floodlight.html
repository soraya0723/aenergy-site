GoodWe DNS G3 Series

The GoodWe DNS G3 series inverter is specially designed for single-phase residential applications. Integrated with high-current input and DC input oversizing capabilities, it ensures optimized power generation and reliable returns. With its lighter, fanless compact design, it runs super quietly (below 25dB) while providing guaranteed safety in extreme environments.

  • ⚡ 150% DC input oversizing
  • ⚡ 110% AC output overloading
  • ⚡ 97.9% maximum efficiency
  • ⚡ <25dB fanless quiet operation
  • ⚡ IP66 ingress protection

Key Features

Superb Safety & Reliability

The integrated safety features of optional AFCI, IP66 protection, and smart SPD failure alarm assure continuous safety.

  • Optional AFCI
  • Optional AC & DC Type II SPD
  • SPD failure alarm

Smart Control & Monitoring

Supports multiple communication protocols including Modbus RTU and Sunspec, plus 24/7 load consumption monitoring with GoodWe HomeKit 1000.

  • Dry contact connection
  • Multiple communication protocols
  • 24-hour load consumption monitoring

Optimal Generation

Compatible with high-power modules, allowing 16A input per string, 150% DC oversizing, and 110% AC overloading for maximum yield.

  • 16A input current per string
  • 150% DC input oversizing
  • 110% AC output overloading
  • 97.9% maximum efficiency
