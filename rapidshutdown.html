GoodWe GM330 Smart Meter

The GoodWe GM330 Smart Meter is a three-phase meter designed to calculate local electricity consumption and demand with high accuracy (Class 0.5). With support for flexible CT inputs, wide voltage range, and export power limiting, the GM330 enables precise load monitoring and efficient integration with hybrid inverters for zero-feed-in applications.

  • ⚡ Accuracy Class 0.5
  • ⚡ Export power limit & load monitoring
  • ⚡ Supports nA:5A CT inputs
  • ⚡ Wide input voltage range
