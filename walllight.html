GoodWe MS G3 Series

The GoodWe MS G3 series inverter (5–10 kW) provides powerful and versatile solutions for residential applications. With 3 MPPTs, it handles complex rooftops efficiently and achieves high power yields. Its ultra-low 50V startup voltage enables earlier power generation each day, while supporting up to 20A DC input current per MPPT makes it ideal for high-power modules.

  • ⚡ 3 MPPTs for complex rooftops
  • ⚡ 50V ultra-low startup voltage
  • ⚡ 20A DC max input per MPPT
  • ⚡ PID recovery function for better module performance
  • ⚡ Compact fanless design for quiet operation
